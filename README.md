# Data Structures and Algorithms Projects

A collection of Java projects focused on implementing core data structures, graph algorithms, custom collections, and runtime-efficient systems without relying heavily on built-in libraries.

These projects emphasize algorithm design, abstraction, performance optimization, and object-oriented programming principles.

---

# Projects

## User Guide Manager
A runtime-optimized user guide management system implemented using skip lists and advanced sorting algorithms.

### Features
- Skip list implementation
- Fast insertion and lookup operations
- Runtime-efficient sorting
- Dynamic data management

### Concepts
- Skip Lists
- Sorting Algorithms
- Runtime Optimization
- Object-Oriented Design

---

## Graph Algorithms and Shortest Paths
Implementation of graph-based algorithms and shortest path computation using custom graph structures.

### Features
- Dijkstra’s shortest path algorithm
- Shortest path tree reconstruction
- Custom graph representations
- Edge list graph implementation

### Concepts
- Graph ADTs
- Priority Queues
- Maps and Sets
- Weighted Graphs
- Pathfinding Algorithms
